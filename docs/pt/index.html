§ 01

O que ele faz

Quatro etapas, um comando. Sem precisar instalar Python. Sem configurar DuckDB. Sem scripts avulsos de PowerShell copiados de um fórum de 2014.

01

Baixa

Puxa os DBCs direto do FTP do DATASUS. Você escolhe UF e intervalo de datas.

02

Converte

DBC → DBF → DuckDB → Parquet, tudo em um processo. Sem CSV intermediário, sem resíduo.

03

Enriquece

Junta códigos IBGE de municípios (5.571 entradas) e mapeamentos CID automaticamente.

04

Expõe

Navegue pela UI, consulte com SQL no DuckDB, ou leia os parquets particionados de qualquer lugar.

§ 02

Para dois públicos

Pesquisadores · clínicos

Instale, clique no atalho, escolha uma pasta, selecione o subsistema e o intervalo de datas. O app baixa e processa tudo localmente. Depois você consulta pela interface web, com filtros e dropdowns — sem SQL.

Veja o tutorial →

Desenvolvedores · times de dados

O mesmo instalador expõe o CLI completo datasus no seu shell (checkbox opcional de PATH). Pipelines automatizados, API Python e DuckDB como superfície de consulta — leia os parquets particionados com polars, pandas, R, o que preferir.

O que esperar
  • — Primeira abertura: ~20s para aquecer os índices parquet.
  • — Instalador sem assinatura. No Windows, um clique em "Mais informações → Executar assim mesmo"; no macOS, "botão direito → Abrir" na primeira vez.
  • — O SIM é publicado com ~2 anos de atraso — o app avisa.
Ficha técnica

Construído com Astro, Tailwind, Fraunces, Public Sans e JetBrains Mono. Sem rastreadores. O único cookie armazena a sua preferência de idioma.

DATASUS é uma marca registrada do Ministério da Saúde do Brasil. Este é um projeto independente, de código aberto, sem vínculo com o DATASUS ou com o Ministério da Saúde.
